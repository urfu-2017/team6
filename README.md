# Kilogram Messenger

### team 6

В корне проекта создать файл .env для переменных окружения
```
NODE_ENV=development
CLIENT_ID=
CLIENT_SECRET=
AUTH_CALLBACK=http://localhost:8080/auth/callback
API_KEY=
```

```
npm install
npm start

SPA: http://localhost:8080
API: http://localhost:8080/api/v1
```
